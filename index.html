MAP672: Programming for Web Mapping

Jack Smarr: Bigtime Mapper and All Seeing Interpreter of Data

Craft Breweries per Population

This map shows the locations of craft breweries in relation to population density. The client wanted to use brewery waste to raise Talapia for restaurants

This map was made using QGIS and a csv file from the American Association of Craft Brewers, along with 2010 County Census Data

  • See the map:Craft Breweries of North America and Hawaii
  • Data Source: 2010 US Census County Population Data
